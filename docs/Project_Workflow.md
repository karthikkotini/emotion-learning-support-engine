# Emotion Detection & Learning Support Engine - Project Workflow

## Workflow

1. User enters login credentials.
2. User selects an academic field.
3. User enters a learning problem or question.
4. The text is preprocessed.
5. The BiLSTM and BERT models analyze the text.
6. The system predicts the primary and secondary emotions.
7. Gemini AI generates an empathetic learning response.
8. The response and emotion scores are stored.
9. Session history is displayed.
10. Analytics dashboard visualizes user interactions.
