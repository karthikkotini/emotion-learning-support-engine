# Emotion Detection & Learning Support Engine

An AI-powered learning assistant that detects student emotions from text input and provides personalized learning support using Machine Learning and Google Gemini AI.

## Project Structure

```
emotion-learning-support-engine/
├── backend/
├── frontend/
├── model/
├── dataset/
├── diagrams/
├── docs/
├── requirements.txt
└── README.md
```

## Features

- Emotion Detection
- BiLSTM & BERT Models
- Gemini AI Integration
- Streamlit Interface
- Emotion History
- Learning Recommendations

## Technologies

- Python
- TensorFlow
- Streamlit
- Google Gemini API
- Transformers
